🌸 Navarah – Maternal Care App

A project developed for Techathon 🚀
Navarah is a maternal healthcare app designed to support expecting mothers and new parents. It provides pregnancy tracking, maternal health tips, appointment reminders, and baby growth monitoring — all in one simple and accessible platform.

✨ Features

🍼 Pregnancy Week Tracker – Personalized insights for every stage.

🩺 Health & Nutrition Guidance – Articles and recommendations for maternal well-being.

📅 Doctor Appointment Reminders – Never miss a check-up.

👶 Baby Growth Monitoring – Track milestones after birth.

📖 Educational Resources – Learn about safe motherhood and childcare.

🛠️ Tech Stack

Frontend: Flutter / React Native

Backend: Node.js / Django / Firebase

Database: Firebase / MongoDB / MySQL

Authentication: Firebase Auth / JWT-based login

🚀 Getting Started

Clone the Repository

git clone https://github.com/vijayaragavanr18/Techathon.git
cd Techathon


Install Dependencies

npm install   # or flutter pub get / pip install -r requirements.txt


Run the App

npm start     # or flutter run / python manage.py runserver
